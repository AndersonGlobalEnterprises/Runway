# AGE Runway

Done-for-you AI content pipeline. Voice clone + script generation + video rendering + multi-platform publishing — automated end-to-end.

## What it does

1. Queue a topic
2. AI generates a script in your voice
3. ElevenLabs clones your voice for audio
4. Creatomate renders a branded video
5. Publishes to LinkedIn, Instagram, X — automatically

## Stack

- n8n Cloud — pipeline orchestration
- ElevenLabs — voice cloning
- Creatomate — video rendering
- Upload-Post API — multi-platform publishing
- Google Sheets — content queue per client
- Portal (age-flowops-portal) — billing + onboarding UI
